Photography Portfolio - Design Plan

Website structure and layout before development

Purpose of the Website

The purpose of this website is to showcase my photography work, share a short biography, and allow visitors to contact me.

Planned Sections

  1. Header with website title and photography quote
  2. Introductory paragraph about photography
  3. Biography section with personal information
  4. Gallery section listing favorite photographs
  5. Contact form for users to send messages
  6. Footer with copyright details

Planned Styling

  • Use of background colors to separate sections
  • Consistent font family across the website
  • Box model properties such as margin and padding
  • Borders around images
  • Hover effects on links using pseudoclasses
